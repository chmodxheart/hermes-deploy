# Terraform Proxmox NixOS LXC Lab

Terraform owns Proxmox-side LXC provisioning in this repo.

This directory owns the Proxmox infrastructure layer for NixOS-targeted LXC containers.
Guest operating system configuration lives in the sibling `../nixos/` tree.
